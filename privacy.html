Privacy Policy

Last updated: 10 June 2026

This static website does not ask for your name, email, phone number, account, or any form submission.

1. Personal Data Collected by This Site

This site does not intentionally collect personal data from visitors. There are no login forms, contact forms, newsletter forms, comments, payment flows, or first-party analytics scripts.

2. Cookies and Tracking

This site does not set first-party cookies and does not include advertising trackers. It uses externally hosted D3.js for the visualization script and is hosted on GitHub Pages.

3. Hosting and Third-Party Services

Because the site is hosted on GitHub Pages, GitHub may process standard technical information such as IP address, browser details, request URLs, and timestamps as part of serving the website and maintaining security. Source links open third-party sites that have their own privacy practices.

4. Data Used in the Visualization

The dashboard uses public labour-market sources and modeled occupation-level estimates. It does not contain personal data about individual workers, students, users, or visitors.

5. Children's Privacy

The site is educational and can be viewed by students, but it does not knowingly collect personal data from children.

6. India Data Protection Context

The privacy posture is designed around data minimization: no visitor personal data is intentionally collected by this static site. India's Digital Personal Data Protection Act, 2023 concerns the processing of digital personal data; this site avoids collecting such data directly.

7. Contact

For privacy concerns or correction requests, use the project repository: GitHub repository.
